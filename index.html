Instructions on how to proceed :

  1. Enter a rational number in the text field with placeholder text as "Enter a rational number" (e.g. 10 or 123.457).
  2. Enter a rational number representing an irrational number upto finite number of digits (e.g. 3.1415926535 for pi).
  3. Press the button with text "raised by"
